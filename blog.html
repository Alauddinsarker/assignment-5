What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a hierarchical representation of a document's structure, allowing programming languages like JavaScript to dynamically access and manipulate elements, attributes, and content in web pages.

How do you select an element from the DOM?

You can select DOM elements using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll() for specific elements or groups matching IDs, classes, or tags.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements. It's useful because it educes the number of event listeners and automatically handles events for dynamically added child elements.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes with the help of some attributes like getAttribute(), setAttribute(), removeAttribute() and Using Classes(classList.add(),classList.remove(),classList.toggle())
